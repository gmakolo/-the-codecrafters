# Base Converter

This is a small Go program that converts numbers between hex, binary, and decimal.

## How to run

1. Make sure Go is installed.
2. Open terminal and go to the folder with the code.
3. Run: go run mami.go

4. The program will ask for input. Type like this: "What would you like to convert?"
EXAMPLES:
convert 1E hex
Decimal: 30

convert 10 bin
Decimal: 2

convert 255 dec
Binary: 11111111
Hex: FF

5. Type "QUIT" to exit.

It also handles wrong input and negative numbers.
