# Why "Pointers"?

The name "Pointers" comes from Go programming.

A pointer stores the address of a value, not the value itself. It helps you find where the real data is.

As a group, we have this name because we will beable to help point each other in the right direction when learning.

We guide, support, and grow together just like pointers help access the right data.

"Pointers" means helping each other find the right path in coding.
