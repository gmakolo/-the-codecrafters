# Introduction to Go

## What I Learned

Go is a programming language used to build software. It is fast and simple to use. It was created by Google.

## Key Points

* It is fast
* It is easy to learn
* It is used to build apps and systems

## Example
To run a Go program we need :
package main
import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}

## My Understanding

Go is a beginner-friendly language that helps you write clean and fast code.
