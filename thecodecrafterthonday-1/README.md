# Simple Calculator

This is a small Go calculator. You can add, subtract, multiply, and divide numbers.

## How to run

1. Make sure Go is installed
2. Open terminal and go to the folder with main.go
3. Run:go run main.go

4. You will see a menu:
1 Add
2 Subtract
3 Multiply
4 Divide
5 Exit
6 Help

5. Type the number for the operation
6. Enter the numbers when asked
7. Type 5 to exit
8. Type 6 to see the help menu

 Example

Enter operation
1 Add
Enter first number
10
Enter second number
6
Result 16
