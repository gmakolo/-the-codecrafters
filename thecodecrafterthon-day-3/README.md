# String Transformer (Go)

## What this program does

This is a small Go program that changes text using different commands.
You type a command and some text, and the program will transform the text.

The program keeps running until you type "exit"

## Commands

* upper → makes all letters UPPERCASE
* lower → makes all letters lowercase
* cap → makes the first letter of every word capital
* title → like cap but small words like a, the, of stay lowercase
* snake → turns spaces into `_` and makes everything lowercase
* reverse → reverses each word
* exit → closes the program

## How to run

Open the terminal in the folder and run:
go run main.go
